# My Website

A responsive, multi-page website template with a professional design and clean navigation structure.

## Project Overview

This website is a modern, fully responsive template built with HTML and CSS. It includes multiple pages with a consistent header navigation menu, footer, and professional styling.

## Features

- ✅ **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- ✅ **Multi-Page Structure** - Separate pages for different sections
- ✅ **Professional Navigation** - Sticky header with navigation menu
- ✅ **Contact Form** - Fully styled contact form for user inquiries
- ✅ **Service & Project Cards** - Grid layout for showcasing services and projects
- ✅ **Modern Footer** - Footer with quick links, social media, and contact information
- ✅ **Smooth Transitions** - CSS animations and hover effects for better UX

## File Structure

```
my-site/
├── index.html          # Home page
├── services.html       # Services page
├── projects.html       # Projects page
├── about.html         # About page
├── contacts.html      # Contact page
├── styles.css         # Main stylesheet
└── README.md          # This file
```

## Responsive Breakpoints

- **Desktop**: Full layout (1200px and above)
- **Tablet**: Adjusted grid layout (768px)
- **Mobile**: Single column layout (480px and below)

## Getting Started

1. **Open the website**: Simply open `index.html` in a web browser
2. **Browse pages**: Use the navigation menu to explore different sections
3. **Fill the contact form**: Use the contact page to send messages (form is styled but needs backend implementation)

## Customization

To customize the website:

1. **Change Site Title**: Edit the title in the logo section of each HTML file
2. **Update Colors**: Modify CSS variables in `styles.css`
3. **Add Content**: Replace placeholder text with your actual content
4. **Add Images**: Create an `images` folder and add image references to HTML files
5. **Modify Navigation**: Edit the navigation menu items in the header
